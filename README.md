## 🛠️ 关于本站的碎碎念（技术底座）

虽然这是一个纯个人使用的博客，但在搭建它时，我还是强迫自己用上了目前前端最新、最硬核的顶配架构：
- **Astro v6**：负责整站极其丝滑的静态打包，实现零 JS 运行，打开速度飞快。
- **Tailwind CSS v4**：基于全新的编译机制打造的纯暗黑科技风设计，带有一些克制的卡片悬浮流光动效。
- **数据解耦**：为了方便我自己以后改网名或者换域名，我把所有的核心名字、Logo 文本全部抽离到了配置文件里，改一次全站同步更新。
