# 🌱 Astro Theme Wutongbins
A minimalist, high-performance, and content-focused digital garden & developer portfolio theme. Fully optimized for the bleeding-edge **Astro v6** and **Tailwind CSS v4**.

一款专为独立开发者、极客量身定制的现代化个人作品集主题。基于最新的 **Astro v6** 与 **Tailwind CSS v4** 架构倾力打造。

---

## ✨ Features / 核心特性

- 🚀 **Astro v6 Content Layer**: Pure Type-safe Markdown management with smooth sidebar rendering. (基于 Astro v6 最新内容层 API，物理隔离博客流与独立页面，消灭类型安全隐患)。
- 🎨 **Tailwind v4 Aesthetic**: Modern dark-mode workspace, featuring glowing cards, glassmorphism UI, and custom animations. (前沿的暗黑科技风设计，集成毛玻璃卡片、动态流光进度条与优雅的微交互)。
- 📄 **Static Page Auto-Direct**: Seamless custom resume/about page driven by Vite's native `import.meta.glob`. (绕过复杂的缓存，利用原生文件流直读 `about.md`，修改简历即时生效)。
- 🏷️ **Industrial Footer**: Standard three-section footer with built-in ecosystem tech badges and a back-to-top shortcut. (规范的大厂三段式自适应页脚，自带生态技术芯片标签与回顶锚点)。
- ⚙️ **Config Driven**: Decoupled site configurations under `src/config.ts` for one-second personal branding synchronization. (完美的数据与视图分离，修改一个配置文件，全站 Logo、版权一秒同步更新)。

## 🛠️ Quick Start / 快速开始

### 1. Clone & Install
```bash
# 克隆本项目
git clone [https://github.com/your-username/astro-theme-wutongbins.git](https://github.com/your-username/astro-theme-wutongbins.git)

# 进入目录
cd astro-theme-wutongbins

# 安装依赖
npm install

# 启动本地开发服务器
npm run dev

Open http://localhost:4321 in your browser to check the magic! 🌟
